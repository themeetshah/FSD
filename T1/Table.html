Table Tags:

  • table (paired)
  • tr (paired)
  • td (paired)
  • thead (paired)
  • tbody (paired)
  • tfoot (paired)
  • caption (paired)

Table Tag Attributes:

  • align (to align table but not the data)
  • bgcolor (for table) (colorname, hexcode, rgb{deprecated})
  • border (pixels)
  • cellpadding (space b/w data and cell) (% or pixels)
  • cellspacing (space b/w 2 cells) (% or pixels)
  • width (table) (% or pixels)
  • height (table) (% or pixels)
  • rules (none, groups, rows, cols, all)

tr Tag Attributes:

  • align (to align table but not the data)
  • bgcolor (for table) (colorname, hexcode, rgb {deprecated})
  • width (% or pixels)
  • height (% or pixels)

td Tag Attributes:

  • align (to align table but not the data)
  • bgcolor (for table) (colorname, hexcode, rgb {deprecated})
  • colspan (no of cols to be merged) (integer)
  • rowspan (no of rows to be merged) (integer)
  • width (% or pixels)
  • height (% or pixels)


Shortcut: table>(tr(th*2))+(tr>(td*2))*3
