Less the generality, less the priority
More specific, more priority 
        
Types of Selectors:

        1. Simple Selectors
        2. Combinator Selectors
        3. Pseudo Class Selectors(Imaginary)
        4. Pseudo Element Selectors(Imaginary)
        5. Attribute Selectors
    
        
Simple Selectors:

        1. Element -> h2, p, div
        2. Id selector -> id="" :it must not start with number
        3. Class selector -> class="" 
            -- it can start with number 
            -- if style conflicting {
                   je class niche hoy e apply thase
               }
            -- id>class
        4. Multiple -> i. Element#id
                       ii. Element.class 
                       iii. Element, Element
        5. * -> Universal
    
        
Combinator Selectors:

        1. Child (>)
        2. Descendent ( )
        3. Adjacent Sibling (+) 
        3. General Sibling (~)  
    
        
Pseudo Class Selectors:

        Syntax: selector:pseudo-class
            -- it must be in either internal or external styling
            -- {maintain the following order}
                i. a : link   
                ii. a : visited   
                iii. a : hover --- div : hover   
                iv. a : active   
                v. input : focus
                vi. input : checked
    
        
Pseudo Element Selectors:

        Syntax: element::pseudo-element
            -- it must be in either internal or external styling
            -- {maintain the following order}
                i. p::first-line
                ii. p::first-letter
                iii. ::before
                iv. ::after
                v. ::marker
                vi. ::selection
    
        
Attribute Selectors:

        1. a[target] {
            color:grey;
           }
        2. a[target:black] {
            color:red;
           }
        3. div[class=^ab] {
            color:red;
           }
